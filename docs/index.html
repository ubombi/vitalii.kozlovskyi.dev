GPG Hardware Key and Git Signing

Setup 2 YubiKey Hardware Keys [K1, K2] Root offline key for CA. Encrypted, backuped and saved in a safe places. subkey S1 for signing + auth -> K1 subkey E1 for encryption _-> K1 subkey S2 for signing + auth _-> K2 subkey E2 for encryption -> K2 Root key is not stored on any of hardware keys.
